Boat Recognition Model

Description

This simple model can recognize 9 types of boats, they include:

  1. Cruise Ship
  2. Ferry boat
  3. Paper boat
  4. Gondola
  5. Buoy
  6. Kayak
  7. Inflatable boat
  8. Freight Ship
  9. Sail boat

You feed the model an image and it will return a list of probabilities. The index of the highest probability is the index of the type that the model believes it to be.

Examples

Tensorflow.js

Loading the model

import * as tf from "@tensorflow/tfjs"

const model = await tf.loadLayersModel('https://raw.githubusercontent.com/SharanSMenon/ml-models/master/boat-type-model/model.json');

Using the model

// This example assumes that you have the model loaded in
// Getting the image
const img = tf.browser.fromPixels(image).resizeNearestNeighbor([250, 250]).expandDims(); // Changing it for the model

// These are the 9 classes, make sure it stays in this order
const classes = ["buoy", "cruise ship", "ferry", "freight ship", "gondola", "inflatable boat", "paper boat", "kayak", "sailboat"];

// Predicting with the image
model.predict(img).data().then(predictions => {
        // Finding the index of the type that the model is most confident.
        // The model returns an array of probabilities, the highest probability is the one that the model thinks it is.
        const index = predictions.indexOf(Math.max(...predictions));
        // Getting the class
        const type = classes[index];
        console.log(type); // Should return "cruise ship" or "gondola" or some other boat type
    }, err => {
        console.log(err)
    })
